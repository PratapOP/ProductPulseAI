# ProductPulse AI

AI-powered Product Management Assistant that transforms startup ideas into actionable business insights.

Built using:

- Flask
- HTML
- CSS
- JavaScript
- Gemini AI
- Render/Vercel Deployment

## Features

### AI Product Analysis

Generate:

- Product Summary
- Problem Statement
- Target Audience
- Market Opportunity
- SWOT Analysis
- Revenue Model
- Competitor Analysis
- Product-Market Fit Score
- Risk Factors
- MVP Roadmap
